# 📊 Superstore Sales Dashboard | Power BI

An interactive business intelligence dashboard built using Power BI to analyze Superstore sales performance, profitability, regional trends, and product category insights.

## 🚀 Features

* KPI cards for Sales, Profit, Orders, and Profit Margin
* Sales trend analysis over time
* Regional filtering using slicers
* Interactive sales map by state
* Top-selling sub-category analysis
* Regional sales distribution visualization

## 🛠 Tools Used

* Power BI
* DAX
* Data Modeling
* Data Visualization

## 📌 Purpose

This project was built to strengthen practical skills in business intelligence, dashboard design, and interactive data storytelling.
